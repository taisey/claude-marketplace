# Create Plugin

Execute in order:

1. **plugin-creation-hear** - ユーザーからworkflowをヒアリング
2. **plugin-creation-design** - タスク分解・スキル設計
3. **plugin-creation-generate** - ファイル生成
4. **plugin-creation-register** - marketplace登録

Principle: 1 Plugin = 1 Workflow. Workflow entry = user-invocable skill, individual tasks = user_not_invoke skills.
