# base

GitHub integration using the `gh` CLI for Claude Code.

## Prerequisites

`gh` CLI がインストール済みであること。未インストールの場合は github skill がガイドを出力します。

## Setup

```bash
gh auth login
```

## Features

| Type | Name | Description |
|------|------|-------------|
| Skill | `github` | GitHub operations via gh CLI (PRs, Issues, Branches, Commits) |
