# base

GitHub integration, PR creation, and connection troubleshooting for Claude Code.

## Setup

### 1. Set GitHub Token

```bash
echo 'export GITHUB_PERSONAL_ACCESS_TOKEN="ghp_your_token_here"' >> ~/.zshrc
source ~/.zshrc
```
