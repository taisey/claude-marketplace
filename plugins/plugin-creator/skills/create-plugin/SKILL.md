# Create Plugin

Execute in order:

1. **hear-workflow** - ユーザーからworkflowをヒアリング
2. **design-plugin** - タスク分解・スキル設計
3. **generate-plugin-files** - ファイル生成
4. **register-plugin** - marketplace登録

Principle: 1 Plugin = 1 Workflow. Workflow entry = user-invocable skill, individual tasks = user_not_invoke skills.
