# Hear Workflow

Ask the user to clarify:

1. **Overall workflow**: What do they want to automate?
2. **Task list**: Individual tasks within the workflow
3. **Shared context**: Knowledge/assumptions needed across all tasks

Keep asking until everything is clear. Leave no ambiguity.
